Colemak

November,7 2022

How did I learn this?

I first discovered this Colemak at Reddit and YouTube. They talked about how the changing to “default” layout will help you in terms of the RSI thing.

It took many months for me to start learn this new layout and finally went full cold turkey mode and first learn about the accuracy. Start slow and slowly build up the speed.

I have learned colemak vanilla and learn the colemak DH mod.

The useful links are:

  • monkeytype.com
  • keybr.com

Practice routine

30 minutes of typing at those mentioned websites.

Things I do like about it

  • Most used keys are right at the home row
  • Make feels good to type with this layout
  • Typing words feel like playing chord
  • Wrist are generally stayed at the same place too

Things I don’t like about it

  • vowels keys are at the right hand
  • typing this with a staggared layout
  • most of the game don’t registered with this layout
